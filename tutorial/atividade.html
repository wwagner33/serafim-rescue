5. Atividade Prática: Hands-on!

Agora que você entende como o código funciona nos bastidores, é hora de sujar as mãos. Como desenvolvedor de jogos, alterar regras e balancear o projeto é a melhor forma de fixar o conhecimento.

Passo 1: Baixar o Código

Abra o seu Terminal ou Prompt de Comando e clone o repositório do jogo usando o Git:

git clone https://github.com/wwagner33/serafim-rescue.git
cd serafim-rescue

Não se esqueça de instalar a biblioteca do Pygame, caso não a tenha: pip install pygame

Passo 2: Missões de Melhoria

Abra o código clonado no seu editor de código favorito (como o VS Code) e tente completar as 3 tarefas abaixo:

🎯 Desafio 1: Balanceamento Extremo

Objetivo: Mude a dificuldade do jogo. Encontre a variável VIDA_BASE_INIMIGO no arquivo principal e aumente-a de 50 para 100. Perceba como a matemática garante que a vida do Jogador (que é 1.5x maior) escale automaticamente.

🎯 Desafio 2: A Nova Arma do Joventino

Objetivo: O seu aliado atual usa um Revolver de 6 balas. Abra o arquivo config.py e adicione uma nova arma ao dicionário ARMAS chamada "Fuzil" (com alto dano e alcance). Depois, no arquivo main.py, altere o atributo self.arma_atual na inicialização da classe Aliado para que ele empunhe essa nova arma no lugar do Revolver.

🎯 Desafio 3: Alteração de Level Design

Objetivo: Crie mais abrigo para se proteger na primeira sala do jogo. Na função gerar_cenario() do arquivo principal, procure a lista chamada obstaculos. Adicione um novo objeto pygame.Rect() a esta lista passando as coordenadas para criar um móvel (quadrado ou retangular) novo dentro da primeira sala.

Boa sorte e bom código! O código é livre, sinta-se à vontade para expandir o jogo como a sua imaginação mandar!
