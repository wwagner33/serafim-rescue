# Serafim Rescue Team

## Descrição
Jogo de tiro onde o jogador controla um time de resgate em missões desafiadoras.

## Características
- Gameplay dinâmico e imersivo
- Múltiplas missões e níveis de dificuldade
- Sistema de equipe customizável
- Gráficos e sons envolventes

## Instalação
```bash
git clone https://github.com/seu-usuario/serafim-rescue.git
cd serafim-rescue
# Siga as instruções específicas de build
```

## Como Jogar
1. Inicie o jogo
2. Selecione uma missão
3. Organize seu time de resgate
4. Complete os objetivos
